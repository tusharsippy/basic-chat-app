# Basic Chat Application using node js and socket.io

## Installation

- git pull https://github.com/tusharsippy/basic-chat-app.git
- cd ./basic-chat-app
- run npm install
- create config.js in public/
- set BASE_URL in config.js
- run index.js using nodemon or pm2
